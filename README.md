# Jonathan - Portfólio

Este repositório contém um site de portfólio para mostrar os projetos que desenvolvo como desenvolvedor web Fullstack.
Vou melhorando e atualizando o site conforme for evoluindo nos estudos.

Estrutura:

- `index.html` — página principal com seções: home, sobre, skills, projetos e contato.
- `css/style.css` — estilos principais.
- `js/script.js` — pequenas interações: menu mobile, rolagem suave e feedback de formulário.

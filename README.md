# Portfólio — Jonathan

Este repositório contém um site estático de portfólio para um desenvolvedor web Fullstack.

Estrutura:

- `index.html` — página principal com seções: home, sobre, skills, projetos e contato.
- `css/style.css` — estilos principais.
- `js/script.js` — pequenas interações: menu mobile, rolagem suave e feedback de formulário.

Personalização rápida:

- Substitua o texto e os projetos em `index.html` pelos seus dados.
- Adicione sua foto em `assets/profile.jpg` e atualize a referência no HTML (ou substitua o placeholder `.photo-placeholder`).
- Configure o formulário: crie um formulário no https://formspree.io/ e substitua `YOUR_FORM_ID` no atributo `action` do formulário.

Como usar localmente:

1. Abra `index.html` no seu navegador (duplo clique) — nenhum servidor é necessário.

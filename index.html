Sobre Mim

Olá! Sou Jonathan Lima, um desenvolvedor Fullstack apaixonado por criar soluções web inovadoras e eficientes. Com sólida experiência no ecossistema Java para o backend e expertise em Angular e React para o frontend, eu transformo ideias em realidade digital.

Meu objetivo é construir aplicações robustas, escaláveis e com uma ótima experiência de usuário, sempre buscando as melhores práticas do mercado e tecnologias de ponta.

Minhas Stacks Principais:

Java Spring Javascript Angular React
